# 6-4 수집된 데이터 압축하여 PC로 이동하기

* 라즈베리파이에 저장된 주행 데이터를 관리하고, 학습 환경으로 옮기는 방법을 학습합니다.

## 데이터압축하기

* 수집된 데이터를 압축해보도록 합니다. 위로 가기 버튼을 눌러 /home/pi 경로로 이동합니다.

```
zip -r video.zip video
```


## PC로 전송하기

* 압축된 파일을 PC로 전송하는 방법에 대해서 알아봅니다.
* VNC뷰터에서 중앙 위쪽에 마우스를 가져다 대면 팝업 메뉴가 생성됩니다.
* 그 중에 파일 전송을 위한 [Transfer files] 부분을 클릭합니다.

```
zip -r video.zip video
```
