# CHAPTER 06 인공지능 자율주행 데이터 수집

* 자율주행 인공지능 학습을 위한 데이터 수집 과정을 학습니다.
* AI 자율주행 시스템을 만들기 위해서는 카메라 영상을 기반으로 한 방대한 학습 데이터가 필요합니다.
* 이를 위해 OpenCV를 활용하여 자동차의 조종 화면을 구성하고, <br>
  주행 중 촬영되는 이미지를 료율적으로 저장하는 방법을 실습합니다.
* 또한 수집된 데이터를 정리하고 압축하여 PC로 옮기는 과정까지 다루며, <br>
   실제 딥러닝 삭습에 활용할 수 있는 데이터셋 구축의 전 과정을 경험합니다.
* 본 장의 실습으로 학습자는 자율주행 학습용 데이터를 직접 생성하고 관리할 수 있는 능력을 갖추게 됩니다.
