# CHAPTER 08 학습 모델 적용 및 자율주행 구현

* 앞에서 학습한 인공지능 모델을 라즈베리파이에 적용하여 실제 자율주행을 구현하는 과정을 학습합니다.
* 앞서 학습된 모델을 불러오고, 필요한 라이브러리를 설치하여 실행 환경ㅇ을 구성한 뒤, 자동차가 카메라 영상을 기반으로 스스로 주행하도록 실습합니다.
* 특히 차선을 인식하고 이를 따라가는 자율주행 기능을 구현하면서 인공지능 모델이 실제 하드웨어와 결합해 동작하는 전 과정을 경함하게 됩니다.
* 이 장의 학습을 통해 학습자는 "AI 모델의 이론-> 학습 -> 실제 적용" 으로 이어지는 완전한 자율주행 시스템의 흐름을 이해하게 됩니다.
