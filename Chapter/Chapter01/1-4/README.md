# 1-4 파이썬 가상환경 구성 및 필수 라이브러리 설치

* 파이썬 가상환경을 구성하여 독립적인 개발 공간을 마련하는 방법을 배웁니다.
* YOLO, OpenCV, PyTorch 등 자율두행 구현에 필요한 필수 라이ㅡ러리를 설치하고, 화경 구성이 오라르게 황료되었는지 확인합니다.
* 이 과정을 통해 학습자를 자율주행 프로그램을 안정적으로 실행할 수 있는 준비를 마치게 됩니다.

<br>

* 라스베리파이의 os인 bookworm이후 버전부터는 보안이 강화되어 기본 설치된 파이썬에 라리브러리 등을 사용자가 추가로 설치하여 사용하지 못하도록 변경되었습니다.
* 파이썬을 사요하면 가살완경을 구성하여 사용할 것을 bookworm 버전부터는 추천하고 있습니다.

<br>
