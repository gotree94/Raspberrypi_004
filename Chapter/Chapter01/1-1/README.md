* 이를 통해 학습 전반에 대한 방향성을 세우고 프로젝트의 전체적인 구조를 파악할 수 있습니다.

## 1) 자율주행 자동차란 무엇인가?

* 자율주행 자동차(Autonomous Vehicle)는 <br>운전자의 직접적인 조작 없이 자동으로 주변환경을 인식하고 주행 경로를 판단하여 스스로 움직이는 자동차를 의마힙니다.

* 즉, 사람이 핸들을 잡지 않아도 자동차가 카메라, 센서, 인공지능 알고리즘을 통해 상황을 판단하고 제어하는 기술이 핵심입니다.

* 자율주행 시스템은 인지(Perception), 판단(Decision), 제어(Control)의 세 단계로 구성됩니다.
* 인지(Perception) - 카메라, 라이다, 초음파 센서 등으로 도로 상황을 파악
    * 카메라, 레이더, 라이다 등의 센서를 이용하여 주변 환경을 인식합니다.
    * 차선, 차량, 보행자, 신호등, 장애물 등을 검출합니다.
* 판단(Decision) - 인공지능 모델이 현재 상황을 분석하고 적절한 주행 전략 결정
    * 인식한 정보를 기반으로 현재 상황을 분석합니다.
    * 차선 변경, 속도 조절, 회전, 정지 등의 주행 계획을 수립합니다.
* 제어(Control) - 모터, 서보모터, 브레이트 등을 제어하려 실제 주행 수행.
    * 조향(Steering)
    * 가속(Acceleration)
    * 제동(Brake)

* 이 교재에서는 라이다나 복잡한 센서 대신 카메라 영상과 딥러닝 인공지능을 중심으로 자율주행을 실습합니다.
* 이를 통해 인공지능이 여상 데이터를 어떻게 해석하고 주행에 활용되는지를 직관적으로 이해할 수 있습니다.

## 2) 자율주행 기술의 발전 단계

자율주행 기술은 미국자동차공학회(SAE) 기준으로 0단계에서 5단계까지로 구분됩니다.

| 단계 | 설명 |
|:--------------------:|:--------------------:|
| 0단계 (수동 운전) | 운전자가 모든 조장 수행 | 
| 1단계 (운전자 보조) | 차선 유지 보조, 자동 감속 등 일부 기능 지원 |
| 2단계 (부분 자동화) | 가속, 제동, 조향을 차량이 보조하지만, 운전자는 항상 개입 필요 |
| 3단계 (조건부 자동화) | 특정 조건에서 차량이 완전 주행 가능, 운전자는 대기 상태 |
| 4단계 (고도 자동화) | 대부분의 환경에서 차량이 자율주행 가능 |
| 5단계 (완전 자동화) | 사람이 전혀 개입하지 않는 완전 자율주행 단계 | 

* 본 교재에서 다루는 내용은 1~2단계 수준의 실습용 자율주행 시스템 입니다.
* 즉, 가람이 초기 제어 명령을 내리더라도 자동차가 카메라 영상과 인공지능을 기반으로 스스로 차선을 인식하고 방량을 제어하는 수준을 구현하게 됩니다.

## 3) 라즈베리파이 기반 자율주행 시스템 구성

자율주행을 구형하기 위해서는 하드웨어와 소프트웨어가 유기적으로 결합되어야 합니다.
이 교재에서는 다음과 같은 구성 요소를 사용합니다.

### 1. 하드웨어
   * 라즈베리파이 (RaspberryPi) : 자동차의 두뇌 역할, 이미지 처리 및 제어 수행
   * 카메라모듈 (RaspberryPi Camera) : 전방 영상을 실시간으로 입력
   * 모터 드라이버 : DC모터의 속도 및 방향 제어
   * 서보모터 : 자동차 앞바퀴의 조향 제어
   * 부저, LED, 버튼 : 사용자 인터페이스 장치
   * 배터리 및 전원 장치 : 차량 구동용 전원 공급
<br><br>
### 2. 소프트웨어
   * 운영체제(OS) : RaspbrryPi Os(리눅스 기반)
   * 개발언어 : Python
   * 라이브러리 : OpenCV(영상처리), PyTorch(딥러닝), GPIOZero(하드웨어 제어)
   * 딥러닝 프레임워크 : YOLOv8, CNN 모델 등

## 4) 자율주행 자동차 학습의 흐름

이 교재에서는 다음과 같은 순서로 자율주행 자동차를 만들어 나갑니다.

* 자동하 조립 및 하드웨어 제어 (1 ~ 3장) : 모터, 서보, LEDm 버튼 등 제어 실습
* 영상 인식 기술 학습(4장) : OpenCV를 이용해 여상 분석, 도형 인식 등 수행
* 인공지능 기초 학습(5장) : PyTorch를 사용한 신경망, 텐서, 자동미분 이해
* 데이터 수지 및 학습 (6 ~ 7장) : 주행 데이터 수집 후 모델 삭습 수행
* 모델 적용 및 자율주행 구현 (8 ~ 9장) : 학습된 모델을 이용해 실제 주행 및 객체 인식 수행

## 5) 학습 목표

이 교재의 학습 목표는 단순한 자동차 제작이 아니라, 인공지능 기반 자율주행 시스템의 이해와 구현입니다.

(1) 지식 목표

   * 자율주행 자동차의 기본 구조와 원리를 이해한다.
   * 컴퓨터 비전과 인공지능의 기본 개념을 학습한다.
   * RaspberryPi 환경에서 하드웨어 제어 방법을 익힌다.

(2) 기능 목표

   * 카메라, 모터, 서보 등 하드웨어 장치를 제어하는 코드를 작성할 수 있다.
   * OpenCV를 이용해 영상 데이터를 처리하고 객체를 인식할 수 있다.
   * PyTorch로 간단한 딥러닝 모델을 학습시키고 결과를 테스트할 수 있다.
   * 학습된 모델을 라즈베리파이에 적용해 실제 자육주행을 구현할 수 있다.

(3) 태도 목표

   * 문제를 단계적으로 해결하며 시스템적으로 사고하는 능력을 기른다.
   * 하드웨어와 소프트웨어의 융합을 통해 창의적인 응용력을 발휘한다.
   * 인공지능 기술의 원리를 실험적으로 이해하고 탐구하는 태도를 기른다.
