# 1-1 자율주행 개요 및 학습 목표

* 자율주행 자동차의 기본 개념과 핵심 기술 요소를 살펴봅니다.
* 자율주행 시스템이 어떻게 인식, 판단, 제어 단계를 거치는지 이해하고, 본 교재를 통해 달성하고자 하는 학습 목표를 명확히 합니다.
* 이를 통해 학습 전반에 대한 방향성을 세우고 프로젝트의 전체적인 구조를 파악할 수 있습니다.

## 자율주행 자동차란 무엇인가?

* 자율주행 자동차(Autonomous Vehicle)는 운전자의 직접적인 조작 없이 자동으로 주변환경을 인식하고 주행 경로를 판단하여 스스로 움직이는 자동차를 의마힙니다.


인공지능 기술은 우리 생활 속에서 빠르게 발전하고 있습니다.

자동차, 의료, 제조, 국방 등 다양한 분야에서 인공지능이 활용되고 있으며, 앞으로도 그 활용 범위는 더욱 확대될 것으로 예상됩니다.

이 중에서도 가장 대표적인 분야가 바로 자율주행입니다.

자율주행(Autonomous Vehicle)은 운전자의 개입 없이 스스로 주변 환경을 인식하고 판단하여 목적지까지 이동하는 기술입니다.

이를 위해서는 다양한 센서와 카메라, 레이더(Radar), 라이다(LiDAR), GPS 등의 정보를 수집하고 이를 분석하는 과정이 필요합니다.

자율주행 시스템은 일반적으로 다음과 같은 세 단계로 구성됩니다.

1. 인지(Perception)
카메라, 레이더, 라이다 등의 센서를 이용하여 주변 환경을 인식합니다.
차선, 차량, 보행자, 신호등, 장애물 등을 검출합니다.
2. 판단(Decision)
인식한 정보를 기반으로 현재 상황을 분석합니다.
차선 변경, 속도 조절, 회전, 정지 등의 주행 계획을 수립합니다.
3. 제어(Control)
조향(Steering)
가속(Acceleration)
제동(Brake)

등의 제어를 수행하여 차량을 안전하게 주행시킵니다.

이러한 과정을 매우 짧은 시간 안에 반복 수행함으로써 차량은 사람처럼 주변 환경을 인식하고 안전하게 주행할 수 있습니다.

## 자율주행 기술의 발전 단계

자율주행 기술은 미국자동차공학회(SAE) 기준으로 0단계에서 5단계까지로 구분됩니다.

| 단계 | 설명 |
|:--------------------:|:--------------------:|
| 0단계 (수동 운전) | 운전자가 모든 조장 수행 | 
| 1단계 (운전자 보조) | 차선 유지 보조, 자동 감속 등 일부 기능 지원 |
| 2단계 (부분 자동화) | 가속, 제동, 조향을 차량이 보조하지만, 운전자는 항상 개입 필요 |
| 3단계 (조건부 자동화) | 특정 조건에서 차량이 완전 주행 가능, 운전자는 대기 상태 |
| 4단계 (고도 자동화) | 대부분의 환경에서 차량이 자율주행 가능 |
| 5단계 (완전 자동화) | 사람이 전혀 개입하지 않는 완전 자율주행 단계 | 

본 교재에서 다루는 내용은 1~2단계 수준의 실습용 자율주행 시스템 입니다.
즉, 가람이 초기 제어 명령을 내리더라도 자동차가 카메라 영상과 인공지능을 기반으로 스스로 차선을 인식하고 방량을 제어하는 수준을 구현하게 됩니다.
