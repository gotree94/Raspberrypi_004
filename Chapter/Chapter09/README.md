# CHAPTER 09 인공지능 객체 검출 Yolov8

* 최신 객체 검출 모델인 YOLOv8(You Only Look Once version 8)을 활용하여 자율주행 자동차에 필요한 객체 인식 기능을 구현합니다.
* YOLOv8은 실시간으로 높은 정확도의 객체 탐지를 수행할 수 있어, 자율주행 시스템의 핵습 기술 중 하나로 널리 사용됩니다.
* 본 장에서는 기본 모델을 활용한 객체 검출부터, 직접 수집한 데이터를 이용한 사용자 맞춤형 학습 모델 제작, 그리고 자율주행 코드에 객체 검출 기능을 결합하는 과정까지 다릅니다.
* 이 과정을 통해 학습자는 인공지능 비전 기술을 실제 자율주행 시스템에 통합하는 방법을 익히고, 주행 안정성을 강화하는 AI 기능을 직접 구현할 수 있습니다.
* 이 장은 "실시간 인식 + 주행 제어"가 결합된 완성형 자율주행 시스템 구현을 목표로 합니다.
