# CHAPTER 01 자율주행 자동차 시작하기

* 이 장에서는 라즈베리파이와 파이썬을 이용하여 자율주행 자동차를 구현하기 위한 첫 간뎨를 다룹니다.
* 자율주행 시스템의 기본 개념을 이해하고, 실제 자동차를 조립하며 개발 환경을 구성하는 과정을 통해 프로젝트의 전체적인 흐름을 파악하게 됩니다.
* 또한 하드웨어와 소프트웨어가 어떻게 결합되어 하나의 자율주행 시스템을 이루는지를 직업 경험할 수 있습니다.
* 본 장의 학습을 통해 이후의 주행 알고리즘 실습을 위한 기초를 다질 것빈디ㅏ.
* 이제 자율주행 자동차의 세계로 첫발을 내디뎌 봅시다.
