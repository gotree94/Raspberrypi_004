# CHAPTER 07 파이토치로 모델 학습하기

* 라즈베리파이에서 수집한 주행 데이터를 활용하여 인공지능 학습모델을 생성하는 과정을 학습합니다.
* 모델 학습은 연산량이 많아서, 더 빠른 처리를 위해 PC 환경에서 PyTorch를 이용하여 실습을 진행합니다.
* Thonny IDE 설치부터 파이썬 가상환경 구성, 필수 라이브러리 설치, 그리고 자율주행 삭습 모델 구현까지 순서대로 다릅니다.
* 이 과정을 통해 학습자는 실제 데이터를 기반으로 AI 모델을 학습시키고, 자율주행 자동차가 스스로 판단하도록 만드는 핵슴 단계를 이해하게 됩니다.
