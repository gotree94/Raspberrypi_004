# CHAPTER 04 OpenCV 배우기

* 컴퓨터 비전 라이브러리인 OpenCV를 활용하여 여상 데이터를 처리하고 분석하는 방법을 학습니다.
* 카메라로 입력된 영상을 기반으로 다양한 객체를 인식하고, 이를 자동차 제어에 활용하는 과정을 단계적으로 실습합니다.
* 영상의 기본 추리 기법부터 시작하여, 바코드-QR코드 검출, 조형 인식, 색상 기반 추적 등 실제 응용 살계를 다릅니다.
* 이 과정을 통해 학습자는 컴퓨터 비전의 기본 원리를 이해하고, 자율주행 시스템의 핵심 기술인 영상 인식 기반 제어의 기초를 익히게 됩니다.
