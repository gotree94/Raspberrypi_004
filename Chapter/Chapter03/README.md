# CHAPTER 03 자동차의 기능 다루기

* 자율주행 자동차를 구성하는 주요 하드웨어 장치들의 기본 제어 방법을 학습합니다.
* LED, 버튼, 부저, DC모터, 블루투스 모듈 등은 자동차의 상태를 표현하고, 외부와 상호작용하는 데 중요한 역할을 합니다.
* 각 장치를 제어하는 실습을 통해 GPIO친의 입력과 출력 원리를 이해하고, 하드웨어 제어의 기초를 다질 수 있습니다.
* 또한 블루투스 통신을 이용하여 자동차를 무선으로 제어하는 방법까지 다루어, 실제 동작할 수 있는 시스템을 구현할 준비합니다.
* 이 과정을 통해 하드웨어 제어 능ㄹ력을 샹항하고, 이후 자율주행 알고이즘과 결합할 수 있는 기반을 마련합니다.
