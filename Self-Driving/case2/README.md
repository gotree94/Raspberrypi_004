## 9. 주요 이상점

1. **자이로 Rate 3370°/s 스파이크** — 중복 timestamp(dt=0)로 인한 나눗셈 오류
2. **Yaw vs Path 각도 상관계수 -0.32** — SLAM 위치와 자이로 헤딩의 불일치
3. **~79% 정지 상태** — 대부분 스캔/계산 대기 시간 (scan-and-move SLAM 패턴)
4. **125초 연속 정지** (t=55.7~180.3s) — 긴 정체 구간
5. **시작/종료점 오차 6.5 cm** — 512cm 주행 후 잔여 오차 (SLAM drift or 실제 오차)

## 10. 활용 방안

### 10.1 자이로 기반 조향 GT
```python
# Gyro_Yaw → steering command 변환
yaw_rate = np.gradient(gyro_yaw, dt)
steering = np.clip(yaw_rate / max_rate, -1.0, 1.0)
```
- `Gyro_Yaw`의 미분(각속도)을 조향 명령으로 매핑 가능
- 장점: 실제 물리 측정, 연속값, case1 gz보다 샘플 많음
- 단점: SLAM X/Y와 상관 낮음, dt=0 아티팩트 필터링 필요

### 10.2 엔코더 기반 속도 GT
- `Encoder_Dist` 미분으로 **직선 속도 GT** 생성 가능
- 조향각과 결합하여 **2D 제어 명령** (speed, steering) 생성

### 10.3 초음파 기반 장애물 회피
- 3개 Sonar (F/L/R) → sparse 270° 거리 맵
- Sentinel 필터링 후 장애물 유무/거리 이진 분류 가능

### 10.4 다중 센서 융합
- 엔코더(속도) + 자이로(각속도) + 초음파(거리) → **EKF/UKF 센서 융합** 벤치마크
- SLAM X/Y가 ground truth 기준으로 제공됨

### 10.5 Dead Reckoning Calibration
- 엔코더 + 자이로만으로 추정한 경로 vs SLAM X/Y 비교
- 바퀴 직경, 휠베이스, 자이로 바이어스 추정

## 11. 전처리 권장사항

```python
# 1. Time → elapsed seconds 변환
t0 = pd.to_datetime(df['Time'].iloc[0])
df['t_sec'] = (pd.to_datetime(df['Time']) - t0).dt.total_seconds()

# 2. Sonar sentinel 필터링
df['Sonar_F'][df['Sonar_F'] >= 998] = np.nan
df['Sonar_L'][df['Sonar_L'] >= 998] = np.nan
df['Sonar_R'][df['Sonar_R'] >= 998] = np.nan

# 3. 중복 timestamp 제거
df = df.drop_duplicates(subset='Time')

# 4. Yaw rate 아티팩트 제거
yaw_rate = np.gradient(df['Gyro_Yaw(deg)'], df['t_sec'])
yaw_rate[np.abs(yaw_rate) > 200] = np.nan

# 5. 이동 구간만 추출
moving = df['Encoder_Dist(cm)'].diff() > 0.5
```
