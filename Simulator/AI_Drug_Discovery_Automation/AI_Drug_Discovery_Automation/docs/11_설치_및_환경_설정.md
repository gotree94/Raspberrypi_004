# 11. 설치 및 환경 설정

## 11.1 시스템 요구사항

- **OS**: Linux (권장), Windows, macOS
- **Python**: 3.8 이상
- **RAM**: 최소 8GB (AlphaFold 사용 시 32GB 권장)
- **GPU**: CUDA 지원 GPU (선택사항, AlphaFold/딥러닝 가속)

## 11.2 설치 방법

### Conda 환경 (권장)

```bash
# 환경 생성
conda create -n drug_discovery python=3.10
conda activate drug_discovery

# RDKit 설치 (conda)
conda install -c conda-forge rdkit

# 필수 패키지 설치
pip install -r requirements.txt
```

### requirements.txt


## 11.3 환경 설정

```bash
# 1. 환경 설정 스크립트 실행
python scripts/setup_environment.py

# 2. (선택) 의존성 설치
python scripts/setup_environment.py --install

# 3. 확인만 하기
python scripts/setup_environment.py --check-only
```

## 11.4 외부 도구 설치 (선택)

### AutoDock Vina
```bash
# Linux
wget https://github.com/ccsb-scripps/AutoDock-Vina/releases/download/v1.2.3/vina_1.2.3_linux_x86_64
chmod +x vina_1.2.3_linux_x86_64
sudo mv vina_1.2.3_linux_x86_64 /usr/local/bin/vina
```

### OpenBabel
```bash
conda install -c conda-forge openbabel
```
