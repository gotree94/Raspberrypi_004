# 9. 자동화 워크플로우

## 9.1 개요

`src/pipeline/` 모듈은 DAG 기반 워크플로우 엔진으로 전체 신약 개발 과정을 자동화합니다.

## 9.2 워크플로우 템플릿

| 템플릿 | 설명 |
|--------|------|
| `full_pipeline` | 종단간 신약 개발 (AlphaFold → 도킹) |
| `docking_only` | 가상 스크리닝 및 도킹 |
| `generation_only` | *De novo* 분자 생성 |
| `screening_pipeline` | 가상 스크리닝 |

## 9.3 파이프라인 실행

```python
from src.pipeline.workflow_manager import WorkflowManager
from src.pipeline.orchestrator import PipelineOrchestrator

# 워크플로우 로드
wf_manager = WorkflowManager()
workflow = WorkflowManager.full_pipeline()

# 오케스트레이터 생성
orchestrator = wf_manager.create_orchestrator(workflow)

# 실행
context = orchestrator.run(
    inputs={"target_sequence": "MVLSPADKTNVKAAW..."},
    pipeline_id="my_pipeline_001",
)
```

## 9.4 CLI 실행

```bash
# 전체 파이프라인
python scripts/run_pipeline.py \
    --target "MVLSPADKTNVKAAWGKVGAHAGEYGAEALERMFLSFPTTKTYFPHF" \
    --workflow full

# 도킹 전용
python scripts/run_pipeline.py \
    --workflow docking \
    --receptor data/proteins/1a42.pdb \
    --ligands data/compounds/library.smi
```
