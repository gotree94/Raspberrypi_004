# 8. 분자 도킹 자동화

## 8.1 개요

`src/docking/` 모듈은 AutoDock Vina를 활용한 분자 도킹을 자동화합니다.

## 8.2 도킹 실행

```python
from src.docking.autodock_vina import VinaDocker, BindingBox

docker = VinaDocker()

# 도킹
binding_box = BindingBox(
    center_x=15.0, center_y=20.0, center_z=10.0,
    size_x=20.0, size_y=20.0, size_z=20.0,
)

result = docker.dock(
    receptor_pdbqt="receptor.pdbqt",
    ligand_pdbqt="ligand.pdbqt",
    binding_box=binding_box,
    ligand_name="aspirin",
    ligand_smiles="CC(=O)Oc1ccccc1C(=O)O",
)

print(f"Best affinity: {result.best_affinity} kcal/mol")
print(f"Number of poses: {result.num_poses}")
```
