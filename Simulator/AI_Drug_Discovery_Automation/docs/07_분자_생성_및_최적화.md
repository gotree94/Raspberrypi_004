# 7. 분자 생성 및 최적화

## 7.1 개요

`src/generation/` 모듈은 VAE, 유전 알고리즘, 단편 기반 설계를 통해 새로운 약물 후보 분자를 생성합니다.

## 7.2 VAE 기반 생성 (SMILES VAE)

```python
from src.generation.vae_model import SMILESVAE, SMILESTokenizer, VAETrainer

# 토크나이저
tokenizer = SMILESTokenizer()

# 모델 생성
model = SMILESVAE(
    vocab_size=tokenizer.vocab_size,
    hidden_dim=256,
    latent_dim=128,
)

# 학습
trainer = VAETrainer(model, tokenizer)
dataset = SMILESDataset(training_smiles, tokenizer)
history = trainer.fit(dataset, num_epochs=100)

# 생성
generated = trainer.generate_valid_smiles(num_attempts=100, temperature=1.0)
```

## 7.3 유전 알고리즘 최적화

```python
from src.generation.molecular_optimizer import GAOptimizer

optimizer = GAOptimizer(
    population_size=100,
    mutation_rate=0.1,
    max_generations=100,
    fitness_function=FitnessFunctions.druglikeness,
)

result = optimizer.optimize(seed_smiles=["CCO", "c1ccccc1"])
print(f"Best: {result.best_smiles} (fitness: {result.best_fitness:.4f})")
```
