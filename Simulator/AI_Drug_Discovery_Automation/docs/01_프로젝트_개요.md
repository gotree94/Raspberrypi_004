# 1. 프로젝트 개요

## 1.1 프로젝트 설명

**AI Drug Discovery Automation Platform**은 실제 AI/ML 기술을 활용하여 신약 개발 전 과정을 자동화하는 종단간(end-to-end) 시스템입니다.

## 1.2 주요 목표

- **AlphaFold 통합**: 단백질 아미노산 서열로부터 3차원 구조 예측 자동화
- **RDKit 기반 분자 모델링**: 화합물 처리, 분석, 유사도 검색
- **ML 기반 ADMET 예측**: 흡수·분포·대사·배설·독성 예측
- **VAE/GA 분자 생성**: 새로운 약물 후보 분자 *de novo* 설계
- **AutoDock Vina 도킹**: 단백질-리간드 결합 친화도 예측
- **DAG 파이프라인**: 종단간 자동화, 체크포인트, 재시작

## 1.3 시스템 구성

```
사용자 입력 (SMILES, FASTA, PDB)
        │
        ▼
┌─────────────────────────────────────┐
│          Pipeline Orchestrator       │
│  (DAG 기반 워크플로우 엔진)          │
└─────────────────────────────────────┘
        │
        ├── AlphaFold ──── 단백질 구조 예측
        ├── Molecular ──── 분자 처리 및 분석
        ├── Screening ──── 가상 스크리닝
        ├── ADMET ──────── 약물성 예측
        ├── Generation ─── 분자 생성 및 최적화
        └── Docking ────── 분자 도킹
```

## 1.4 기술 스택

| 영역 | 기술 |
|------|------|
| 구조 예측 | AlphaFold2/3 |
| 분자 모델링 | RDKit |
| 기계 학습 | PyTorch, scikit-learn, XGBoost |
| 분자 도킹 | AutoDock Vina |
| API 서버 | FastAPI + WebSocket |
| 시각화 | matplotlib, py3Dmol |
| 설정 관리 | YAML/JSON + 환경변수 |
