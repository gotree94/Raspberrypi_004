# System Architecture

## 1. 개요

AI Drug Discovery Automation Lab은 **Agent-Based Architecture**를 채택하여 각 AI 기능을 독립적인 TCP 서버 에이전트로 구현합니다. 각 에이전트는 고유한 GUI를 가지며, 실시간으로 내부 상태와 성능 지표를 시각화합니다.

## 3. 에이전트 구조

### 공통 아키텍처

각 에이전트는 다음 컴포넌트로 구성됩니다:

```
┌──────────────────────────────────────┐
│           Agent GUI (tkinter)         │
│  ┌─────────────┐  ┌───────────────┐  │
│  │ Motor Panel  │  │ Sensor Panel  │  │
│  │ (계산 단계)   │  │ (품질 지표)   │  │
│  ├─────────────┤  ├───────────────┤  │
│  │ · MSA Build │  │ · pLDDT: 92.3│  │
│  │ · Inference │  │ · PAE: 5.1   │  │
│  │ · Relax     │  │ · Coverage   │  │
│  └─────────────┘  └───────────────┘  │
│  ┌──────────────────────────────────┐│
│  │        Visualization Area        ││
│  │ (구조/그래프/히트맵 표시)         ││
│  └──────────────────────────────────┘│
│  ┌──────────────────────────────────┐│
│  │        Command Log               ││
│  └──────────────────────────────────┘│
└──────────────────────────────────────┘
         │
         ▼
┌──────────────────────────────────────┐
│      TCP Server (threading)          │
│  · recv() → parse → execute → send() │
└──────────────────────────────────────┘
```

### Computational Motor (컴퓨테이셔널 모터)

물리적 장비의 모터에 대응되는 개념으로, AI 모델의 각 실행 단계를 나타냅니다.

| Motor 타입 | 상태 | 설명 |
|---|---|---|
| IDLE | 대기 | 명령 대기 중 |
| RUNNING | 실행 중 | 현재 해당 단계 실행 |
| COMPLETED | 완료 | 실행 완료 |
| FAILED | 실패 | 오류 발생 |

### AI Sensor (AI 센서)

물리적 장비의 센서에 대응되는 개념으로, AI 모델 결과의 품질 지표를 나타냅니다.

| Sensor 타입 | 범위 | 설명 |
|---|---|---|
| pLDDT | 0-100 | AlphaFold 예측 신뢰도 |
| Binding Affinity | -15~0 kcal/mol | 분자 결합 친화도 |
| QED Score | 0-1 | 약물 유사도 |
| LogP | -2~10 | 지용성 |
| Toxicity Risk | 0-100% | 독성 위험도 |

## 4. 통신 프로토콜

### 명령어 형식

```
COMMAND:param1:param2:...
```

### 응답 형식

```
SUCCESS:데이터
ERROR:오류메시지
TIMEOUT
```

### 공통 명령어

| 명령어 | 설명 | 응답 예 |
|---|---|---|
| STATUS | 에이전트 상태 조회 | `SUCCESS:{"motors":{...},"sensors":{...}}` |
| HELP | 지원 명령어 목록 | `SUCCESS:PREDICT,DOCK,SCREEN,...` |
| RESET | 에이전트 초기화 | `SUCCESS:RESET_DONE` |

## 5. Closed-Loop 워크플로우

```
[시작] 표적 단백질 시퀀스 입력
    │
    ▼
┌─────────────────────────────────────┐
│  AlphaFold: PREDICT:<sequence>      │
│  → 3D 구조 + Confidence Metrics     │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  Virtual Screener: SCREEN:<smiles>  │
│  → Top-N 화합물 리스트              │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  Docking: DOCK:<smiles>             │
│  → Binding Affinity + Pose         │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  ADMET: PREDICT:<smiles>            │
│  → ADMET Properties                 │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  Molecule Generator: OPTIMIZE       │
│  → Optimized Molecules              │
└─────────────────────────────────────┘
    │
    ▼
[반복] 최적화된 분자 재도킹 (Convergence까지)
```
